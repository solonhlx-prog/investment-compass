> 基于公开投资策略知识体系提炼

## 零、一句话定义

**投研罗盘是一套将机构级量化对冲思维，降维适配为中国A股散户可执行的生存-盈利操作系统。**

---

## 一、拆解到底层：A股的三个第一性事实

### 事实1：A股是一个「波段震荡市」，不是单边牛市

不用3000点到6000点。从3000点砸到2600，再从2600拉到3300，一年就结束了。这就是A股的盈利模式。大机构不需要长牛，只需要波动。

**推论**：指望等牛市来赚钱 = 等一个可能十年不来的东西。必须建立适应波动的策略。

### 事实2：市场参与者天然不对等

| 参与者 | 工具 | 信息 | 时间约束 |
|--------|------|------|----------|
| 国家队 | 无限子弹+政策协同 | 先知 | 无 |
| 麒麟会 | 上市公司捆绑+五年代价 | 内幕级 | 无 |
| 量化 | T+0+算法+速度 | 实时数据 | 无 |
| 公募/险资 | 巨量资金+投研 | 机构级 | 考核周期 |
| 游资 | 资金集中+情绪操控 | 中上 | 有 |
| **散户** | **无** | **末位** | **无** |

**散户唯一的优势**：没有资金成本，时间站在你这边。机构有赎回压力、考核周期、排名竞争。你可以等三年，机构等不了。

### 事实3：K线的本质是多空当天打出的「领地范围」

一根K线的最高价 = 多头今天攻到的最远边界，最低价 = 空头今天砸到的最深位置。实体大小不重要，**范围**才重要。多根K线连续是否不断拓高（高点抬高、低点抬高）→ 这就是趋势的唯一定义。

## 三、系统的三层架构

### 第一层：择时 — 判断战场

**功能**：回答「现在是什么市场？」

- 基本面择时：宽松周期？紧缩周期？GDP往上还是往下？
- 资金面择时：有没有增量资金？国家队在不在干活？
- 技术面择时：BBI走平还是拐头？大盘在区间内还是突破？

**输出**：仓位上限。震荡市≤50%，单票≤5%。牛市确认后才加仓。

### 第二层：量化对冲（配置） — 排兵布阵

**功能**：每个票在组合里有明确的角色，不是瞎买。

用投资组合角色分工类比：
- **防守型资产**：低波红利、大消费白酒——对冲市场系统性风险，防守反击
- **进攻型资产**：科技票——市场有赚钱效应时负责冲锋
- **弹性资产**：交易性机会——小仓位博超额，严格止损

**核心逻辑**：不靠单一票暴富，靠组合降低波动 + 确保永远不下牌桌。2000万以下也要配置，因为市场时刻在变化，单轮一个票赚的钱未来一定亏回去。

### 第三层：风控 — 纪律

**四步原则**：不追、不动、不慌、不乱试。

- **不追**：永远等B1买点，不等B2追高
- **不动**：底部区间拿住，不因一根阴线恐慌
- **不慌**：不到7%不算崩，不到连续两个跌停不算系统性风险
- **不乱试**：不在不懂的票上乱试

**止损铁律**：每次交易的止损位在入场时就定好。止损出来只是三五个点的事，但如果单票只占10%仓位，对总净值影响只有0.3%-0.5%。

## 四、核心战法：波段纪律法

**本质**：A股散户唯一能长期存活的交易策略。两个词说清全部逻辑——**波段**（适应A股震荡市，低买高卖）+ **纪律**（策略的核心不是选股，是执行）。

### 四个为什么

1. **为什么适合新手**：不需要盯盘，不需要判断日内波动，收盘看一眼就行
2. **为什么适合A股**：A股的盈利模式是波动不是单边，波段纪律法就是低买高卖波段
3. **为什么牛市不踏空**：永远有底仓（40%红利+消费），不怕突然拉升
4. **为什么熊市不被套**：严格止损 + 分批放飞，涨了先卖一半锁定利润

### 操作要点

- B1买点入场（底部信号确认后）
- 上涨后先放飞一半（锁利+降本）
- 止损设在入场K线最低价往下2-3个价位
- 永远不等B2追高——现阶段没有到做突破的时候

## 五、知识传递体系

投研罗盘不仅是一套策略，更是一个**入门工具**：
- **入门定位**：代表适合无基础的小白，也代表建立个人交易系统之后要有选择的舍弃这套系统。正如《金刚经》所说：“知我说法，如筏喻者，法尚应舍，何况非法”。
- **知识分层**：K线基础 → 均线 → 核心指标（BBI/KDJ/MACD） → 构建个人交易系统 → 高级战法

核心理念：**先扎马步，再学招式，然后建立适合自己的体系。**
