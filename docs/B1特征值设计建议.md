# B1 特征值设计建议 — 基于第一性原理

> 2026-07-07 | 基于10个完美案例验证 + 门槛悖论发现 + 两套系统对比

---

## 一、第一性原理回顾

B1 的底层逻辑是**三层周期**：

```
阳极启升 → 庄家放量突破黄线/白线，启动拉升
敛阴守衡 → 拉升到高点后，有序缩量回调，回踩支撑
归元合一 → 在支撑位附近缩量企稳，J值回到低位，K线窄幅震荡
```

**核心洞察**（来自门槛悖论验证）：
- 阈值型评分（达到门槛=满分）→ 大量股票同分 → 无区分度
- 需要**连续型评分**（越接近完美越高分）+ **区分特征**（打破同分僵局）

## 二、特征值分类体系（四层金字塔）

```
                    ┌──────────────┐
                    │ ④ 区分特征层  │  ← 打破同分，排序用
                    │   (连续型)    │
                   ┌┴──────────────┴┐
                   │ ③ 当前状态层  │  ← 选股日的即时状态
                   │   (连续型)     │
                  ┌┴───────────────┴┐
                  │ ② 回调路径层   │  ← 回调过程的质量
                  │   (连续型)      │
                 ┌┴────────────────┴┐
                 │ ① 突破启动层    │  ← 突破的质量和强度
                 │   (连续型)       │
                 └─────────────────┘
```

## 三、18个特征值详细设计

### 第一层：突破启动层（3个特征，权重30%）

这层衡量"庄家是不是真的在拉"，不是所有放量都是真突破。

| # | 特征名 | 含义 | 类型 | 完美案例范围 | 连续评分公式 | 说明 |
|---|--------|------|------|:----------:|------------|------|
| F1 | **放量突破比** | 突破日/次日最大量 ÷ 前日均量 | 连续 | 1.73~7.11 | `min(30, ratio×6)` | 比值越大，突破越强。乘以6使7.11x≈43分(截断30) |
| F2 | **突破类型** | 黄线突破 vs 白线突破 vs 无突破 | 分档 | 黄线突破 | 黄线=15, 白线=10, 无=0 | 黄线突破 > 白线突破，反映主力决心 |
| F3 | **累计涨幅** | 突破点到最高点的涨幅% | 连续 | 17~233% | `min(15, gain×0.3)` | 涨幅太小(无行情)和太大(炒作)都扣分，最优区间15-50%得满分 |

### 第二层：回调路径层（5个特征，权重30%）

这层衡量"回调是不是有序的"，烂股回调是杂乱无章的。

| # | 特征名 | 含义 | 类型 | 完美案例范围 | 连续评分公式 | 说明 |
|---|--------|------|------|:----------:|------------|------|
| F4 | **顶部缩量比** | 顶部3日均量 ÷ 上涨期最大阳量 | 连续 | 0.26~0.95 | `20 × (1 - max(0, ratio-0.2)/0.8)` | 0.2以下=20分，0.95≈1分。越缩越好 |
| F5 | **回调深度比** | 最高点到选股日跌幅 ÷ 累计涨幅 | 连续 | 0.15~0.75 | `20 × (1 - abs(depth-0.4)/0.5)` | 最优回调约40%的涨幅(浅了没洗够，深了破位) |
| F6 | **支撑偏离度** | 选股日收盘价相对黄线的偏离% | 连续 | -2%~+5% | `20 × (1 - abs(deviation)/10)` | 越贴近黄线越高分，偏离10%=0分 |
| F7 | **缩量趋势斜率** | 回调阶段每日成交量的线性回归斜率 | 连续 | 负斜率 | `20 × min(1, abs(slope)/0.03)` | 量能递减越规律越高分，不递减=0分 |
| F8 | **堆量比** | 上涨阶段累计阳量 ÷ 累计阴量 | 连续 | 1.24~3.18 | `min(15, ratio×5)` | 阳量远大于阴量=筹码在集中，1.24≈6分，3.18≈16分 |

### 第三层：当前状态层（5个特征，权重20%）

这层衡量"现在是不是合适的买点"。

| # | 特征名 | 含义 | 类型 | 完美案例范围 | 连续评分公式 | 说明 |
|---|--------|------|------|:----------:|------------|------|
| F9 | **J值评分** | KDJ的J值（连续而非阈值） | 连续 | -13.4~+13.7 | `20 × (1 - max(0, j+20)/50)` | J=-13→20分, J=0→12分, J=14→6分, J=50→0分 |
| F10 | **小阴小阳度** | 选股日涨跌幅绝对值% | 连续 | 0.3~2.0% | `20 × (1 - deviation/5)` | 0%=20分, 2%=12分, 5%=0分 |
| F11 | **选股缩量比** | 选股日量 ÷ 上涨期最大阳量 | 连续 | 0.13~0.30 | `20 × (1 - ratio/0.8)` | 0.13≈17分, 0.30≈13分 |
| F12 | **金叉后经过天数** | 最近一次白线金叉黄线到选股日的天数 | 分档+连续 | 8~55 | 见下方详表 | 按斐波那契窗口评分 |
| F13 | **J值清洗深度** | 金叉后J值最大值 - 选股日J值 | 连续 | 50~120 | `min(10, delta/10)` | J从高位回落到低位的幅度，越大越好 |

### 第四层：区分特征层（4个特征，权重20%）

这层是**打破同分僵局**的关键——当基础层+路径层+状态层得分相同时，用这层排序。

| # | 特征名 | 含义 | 类型 | 来源 | 说明 |
|---|--------|------|------|------|------|
| F14 | **沙漏审美分** | 5因子图形质量评分 | 0-100连续 | 投研罗盘已有 | 缩量收敛+枢轴邻近+量能斜率+均线结构+事件风险 |
| F15 | **MDC置信度** | 7层多维验证置信度 | 0.1-0.98连续 | 投研罗盘已有 | 麒麟阶段+布林下轨+资金流+RSI6+ADX+缩量+绿砖 |
| F16 | **蜈蚣图分** | K线混乱度评分 | 0-100硬过滤 | 投研罗盘已有 | ≥60分一票否决，<60按分数反向排序 |
| F17 | **麒麟阶段** | 庄家四阶段识别 | 分档 | 投研罗盘已有 | 吸筹>回落>拉升>派发 |

## 四、评分公式

### 总分计算

```
总分 = 基础分(80分) + 区分分(20分)

基础分 = F1×0.10 + F2×0.05 + F3×0.05 + F4×0.10 + F5×0.10
       + F6×0.05 + F7×0.05 + F8×0.05 + F9×0.10 + F10×0.10
       + F11×0.05 + F12×0.10 + F13×0.05

区分分 = F14×0.30 + F15×0.30 + F17×0.20 - F16_normalized×0.20
       (归一化到0-20分范围)
```

### 硬过滤（一票否决）

| 条件 | 逻辑 |
|------|------|
| 蜈蚣图F16 ≥ 60 | 烂股形态，直接排除 |
| F2突破类型 = 无突破 | 没有突破，直接排除 |
| J值 > 14 | 不超卖，直接排除 |
| 白线 < 黄线 | 空头形态，直接排除 |

## 八、总结

> **从5个阈值型特征 → 17个连续型特征的四层金字塔。**

核心原则：
1. **所有主要特征用连续评分**，不用阈值"达到即满分"
2. **硬过滤只保留4个**：无突破排除、J>14排除、白<黄排除、蜈蚣图≥60排除
3. **区分特征层用于打破同分僵局**：沙漏审美+MDC置信度+麒麟阶段
4. **13个基础特征决定"能不能进门"**，4个区分特征决定"排在第几名"
