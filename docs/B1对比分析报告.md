## 一、第一性原理：B1 的本质是什么

B1 不是「找超卖股」，也不是「找低点」。B1 的本质是：

> **找到庄家完成一轮「吸筹→拉升→洗盘→缩量回踩」后，在白线/黄线支撑位附近缩量企稳的买点。**

从这个本质出发，B1 完美图的三个阶段对应「九极归一」：

```
阳极启升 — 庄家放量突破黄线，启动拉升（放量突破 + 累计涨幅 + 堆量比）
敛阴守衡 — 拉升到高点后有序缩量回调（顶部缩量 + 回调量比 + 支撑）
归元合一 — 回到支撑位，缩量企稳，J值低位，小阴小阳（J值 + 小阴小阳 + 选股缩量）
```

**核心洞察**：B1 评的不是「当前便不便宜」，而是「整个走势路径像不像完美图」。路径质量 > 当前状态。

## 四、核心差距总结

### 4.1 投研罗盘 B1 的根本问题

**问题一：J值门槛方向性错误**

投研罗盘设 J≤-8 作为硬门槛，但 10 个完美案例中 8 个 J 值 > 0。这意味着投研罗盘会**过滤掉 80% 的完美图买点**。

Z哥的 J≤14 不是在找「超卖」，而是在找「J值回到低位区」——这个低位是相对前期 J 值高点而言的。赛特新材完美案例 J 峰值 104 → 选股日 1.01，是「从高位回到低位」而非「绝对超卖」。

**问题二：缺少「路径质量」评分**

Z哥 B1 评的是整条走势路径：放量突破 → 拉升 → 顶部缩量 → 回调 → 企稳。每个阶段都有对应的特征评分。

投研罗盘只看当前快照：J值、量比、乖离、双线方向。不关心「这个低点是怎么来的」。

**问题三：缺少「金叉后天数」分类**

Z哥按斐波那契数列（8/13/21/34/55）将 B1 分为不同策略变体，每个变体对应不同的建仓周期。这是 B1 完美图的核心分类维度。

投研罗盘没有这个概念，无法区分「金叉后一周的缩量回踩」和「金叉后两个月的缩量回踩」。

**问题四：缺少「量能相对关系」评分**

Z哥的三个量能特征（④顶部缩量、⑨选股缩量、③堆量比）都是相对于「上涨期最大阳量」计算的，衡量的是「回调过程中量能萎缩的程度」。

投研罗盘的量比是「当日量/前5日均量」，只看短期相对量能，不看相对于整个上涨周期的缩量程度。

### 4.2 投研罗盘 B1 的独有优势

| 优势 | 说明 |
|------|------|
| 蜈蚣图检测 | 5因子检测烂股形态（长影线/十字星/量能无规律/价格无趋势），Z哥没有 |
| 沙漏审美 | 5因子评分图形质量（缩量收敛/枢轴邻近/量能斜率/均线结构/事件风险），Z哥没有 |
| 麒麟阶段 | 4阶段庄家识别（吸筹/拉升/派发/回落），Z哥没有 |
| MDC七层验证 | 多维置信度打分（麒麟+布林+资金流+RSI+ADX+缩量+绿砖），Z哥没有 |
| 参数可配置化 | 12个B1_*环境变量，Z哥是硬编码在notebook中 |
| 平台无关 | 纯Python函数，可在任何Agent框架运行 |

## 五、提升方案（按优先级）

### P0：补齐核心评分特征（紧急）

这是最根本的差距——投研罗盘缺少 6 个 Z哥核心特征。

| # | 特征 | 满分 | 实现要点 | 完美案例参考值 |
|---|------|:---:|---------|:-----------:|
| 1 | **放量突破** | 15 | 找黄线/白线突破点，计算突破日+后2日最大量/前日量 | 2.0~7.11x |
| 2 | **金叉后天数** | 20 | 找最近一次白线金叉黄线，计算到选股日的天数，按斐波那契窗口评分 | 8/13/21/34/55±3 |
| 3 | **顶部缩量** | 15 | 找近期最高点，计算顶部3日均量/上涨期最大阳量 | 0.33~0.90 |
| 4 | **选股缩量** | 10 | 选股日量/上涨期最大阳量 | 0.13~0.30 |
| 5 | **小阴小阳** | 10 | 选股日涨跌幅+振幅+影线比，窄幅震荡满分 | 全部满分 |
| 6 | **堆量比** | 10 | 上涨阶段累计阳量/累计阴量 | 1.24~3.18 |

### P1：调整 J 值逻辑（高优先级）

| 当前 | 目标 | 理由 |
|------|------|------|
| J≤-8 硬门槛 | J≤14 硬门槛（对齐Z哥） | 完美案例80%的J值>0 |
| J钝化排除（min(J)<-3则排除） | J连续负天数≥2（纳刀式/微风式）或≥0（韵升式） | Z哥要求J曾在低位，不是排除 |
| J分：J≤-12→3分, J≤-10→2分 | J分：J<0→9-10分, J<14→8分, J<50→5分 | J是评分维度而非超卖指标 |

### P2：加入减分项体系（中优先级）

| 减分项 | 扣分 | 检测逻辑 |
|--------|:---:|---------|
| ST股票 | -10 | 股票名称包含ST |
| 缩量涨停 | -10 | 涨幅≥9.8%且量比<0.5 |
| 上涨期最大阴量 | -10 | 上涨阶段最大成交量日为阴线 |
| 股价低于黄线×0.98 | -10 | 收盘价 < 黄线 × 0.98 |
| 最大量日上影线过长 | -10 | 上影线 > 柱体1倍 |
| 顶部K线上影线过长 | -10 | 顶部K线上影线 > 柱体1倍 |

### P3：加入卖出体系（中优先级）

Z哥验证过的最优卖出参数（纳刀式 B1-1）：

```
止损：买入日最低价 × 0.97，前3天监控
止盈：盈利>8%触发，卖出1/3
动态止盈：盈利>10%启用，峰值回撤>5%清仓
白线支撑：收盘价连续3天 < 白线×0.98，第4天起检查
```

### P4：保留并融合投研罗盘独有优势（低优先级）

将蜈蚣图、沙漏、麒麟阶段、MDC验证作为「第二层过滤」：
- 第一层：Z哥九极归一评分（≥85分通过）
- 第二层：投研罗盘硬过滤（蜈蚣图/沙漏一票否决）
- 第三层：MDC置信度加分（0.1~0.98作为排序依据）

### P5：回测验证（必须）

用 2025 全年数据回测，构建策略网格：
- 评分门槛 ∈ {85, 90, 95}
- 止盈触发 ∈ {6, 8, 12}
- 动态回撤 ∈ {3, 5, 8, 10}
- 白线连续天数 ∈ {2, 3}

目标：年化≥50% + 最大回撤≤-15%

## 七、结论

投研罗盘的 B1 和 Z哥的 B1完美图是**两套完全不同的系统**：

- **Z哥 B1** = 图形匹配系统（9特征衡量走势路径与完美图的相似度）+ 完整卖出体系 + 回测验证
- **投研罗盘 B1** = 防守过滤系统（硬过滤排除烂股 + MDC多维验证）+ 无卖出体系 + 无回测

投研罗盘在「防守端」（蜈蚣图/沙漏/MDC/麒麟）有独有优势，但在「进攻端」（路径质量评分/金叉分类/量能相对关系）存在 **6 个核心特征完全缺失**。

**最致命的问题**是 J≤-8 的门槛会过滤掉 80% 的完美图买点。这不是参数调优的问题，是评分哲学的方向性偏差——B1 的核心不是「找超卖」，而是「找完美图形路径」。

**提升路径**：P0 补齐核心特征 → P1 修正J值 → P2 减分项 → P3 卖出体系 → P4 融合优势 → P5 回测验证。
