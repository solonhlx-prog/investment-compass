## 三、Z哥系统为何也只通过 5/10？

Z哥系统不是100%能选出所有"完美案例"，原因有三：

### 3.1 后5个案例的评分瓶颈

| 案例 | 瓶颈特征 | 说明 |
|------|---------|------|
| 国轩高科 | F4小阴小阳=12 | 选股日K线振幅偏大，不够"窄幅" |
| 野马电池 | F1放量突破=12 | 突破日放量仅2.0x（完美线≥3x得20分） |
| 光电股份 | 减分-10 | 触发了某项减分（可能是上影线过长） |
| 新瀚新材 | F1放量突破=6 | 突破日放量不够，F1只得6分 |
| 昂利康 | F4小阴小阳=6 | 选股日涨跌幅1.5%，超出±2%窄幅区间 |

### 3.2 "完美案例"的定义问题

这10个案例是**事后回看走势完美**的股票，但不一定在**选股当天**评分就最高。有些案例可能是：
- 选股当天评分不够，但后续走势验证了形态
- 评分体系针对前5个案例（华纳/韵升/微芯/方正/澄天）校准更充分
- 后5个案例的某些特征值偏离了评分体系的"满分区间"

### 3.3 策略分窗的局限性

Z哥按斐波那契数列（8/13/21/34/55）分窗，但10个案例的"金叉后天数"不一定恰好落在某个窗口的满分区间。例如国轩高科在韵升式得84分（F3=20满分），但F4只有12分，拖累了总分。

## 四、两套系统的核心差距

```
投研罗盘 B1：找"超卖+缩量"的股票 → 当前快照状态
Z哥 B1完美图：找"走势路径像完美图"的股票 → 整条路径质量

验证结果：
  投研罗盘 0/10 → 评分哲学方向错误，完美案例根本进不了门
  Z哥      5/10 → 评分体系有效但非万能，后5个案例的某些特征偏离满分区间
```

### 投研罗盘的根本问题

| 问题 | 影响 | 完美案例数据 |
|------|------|-------------|
| J≤-8 硬门槛 | 过滤80%完美案例 | 10个中8个J>0 |
| 量比<0.85 | 过滤50%完美案例 | 完美案例量比0.47~1.23 |
| 乖离MA20≤-3% | 过滤70%完美案例 | 完美案例乖离-6%~+7% |
| J钝化排除 | 排除J确实为负的案例 | 3个J<0的案例被排除 |
| 无"路径质量"评分 | 无法识别完美图形 | Z哥靠5特征×20分衡量路径 |

### Z哥系统的优势与不足

**优势**（投研罗盘完全缺失）：
- 5特征×20分=100分的路径质量评分体系
- 金叉后天数分窗（斐波那契数列）
- 量能相对关系（顶部缩量/选股缩量/堆量比）
- 减分项体系（6项-60分）

**不足**：
- 后5个完美案例在选股日当天评分不够85分
- 评分体系针对前5个案例校准更充分
- F1（放量突破）和F4（小阴小阳）是后5个案例的主要瓶颈

## 五、结论

### 能否选出10个完美案例？

| 系统 | 答案 | 原因 |
|------|------|------|
| 投研罗盘 B1 | **不能（0/10）** | J≤-8门槛方向性错误 + 缺少路径质量评分 |
| Z哥 B1完美图 | **部分能（5/10）** | 前5个案例能选出，后5个评分不够85分 |

### 提升方向

1. **投研罗盘必须重构B1**：从"超卖过滤"转向"路径质量评分"，J门槛从-8放宽到14
2. **Z哥系统也需优化**：后5个案例的F1（放量突破）和F4（小阴小阳）评分门槛可能偏严
3. **两套系统融合**：投研罗盘补齐Z哥的5特征评分 + 保留蜈蚣图/沙漏/MDC防守优势
