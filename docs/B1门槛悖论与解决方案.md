## 一、核心发现：门槛悖论

验证结果揭示了一个**根本性悖论**：

| 评分体系 | 10个案例全部通过 | TOP20区分度 |
|---------|:---:|:---:|
| **放宽门槛**（让10个都通过） | ✅ 10/10通过 | ❌ 0/10进TOP20 |
| **Z哥原始门槛**（≥85分） | ❌ 5/10通过 | ❌ 3/10进TOP20 |

**放宽门槛后，完美案例反而排名更低**——因为大量非完美股票也得了满分，把完美案例淹没了。

## 三、为什么"放宽门槛"反而更差？

### 悖论的本质

```
门槛严 → 完美案例进不来（0/10通过）
门槛宽 → 完美案例进来了，但被淹没（0/10进TOP20）
```

**根因**：5特征×20分的评分体系，每个特征都是**阈值型评分**（达到阈值→满分），不是**连续型评分**（越接近完美越高分）。

当阈值放宽后：
- F1放量突破：≥1.5x→20分 → 大量股票突破日量比>1.5，都得20分
- F2顶部缩量：≤0.50→20分 → 大量股票顶部缩量<0.5，都得20分
- F4小阴小阳：<3%→20分 → 大量股票当天小幅震荡，都得20分
- F5选股缩量：≤0.35→20分 → 大量缩量股票都得20分

**结果**：几百只股票5项全满分=100分，完美案例无法脱颖而出。

### Z哥原始评分稍好但仍不够

Z哥原始评分门槛更严（F1≥3x→20分，F4<2%→20分），所以满分股票少一些（5-23只≥95分 vs 25-316只=100分），但：
- 微芯生物 #60/2334 — 差得太远
- 澄天伟业 #51/658 — 也不在TOP20
- 国轩高科 #294/1860 — 完全淹没

## 四、从第一性原理看解决方案

### 4.1 当前评分体系的根本缺陷

| 缺陷 | 说明 |
|------|------|
| **阈值型评分无区分度** | 达到阈值=满分，超不超阈值是二值的，没有"多接近完美"的概念 |
| **5个维度不够** | 只有5个特征，维度太少，容易大面积撞分 |
| **缺少"图形相似度"** | 10个完美案例的共同点是"图形长得像"，但评分只看数值不看形状 |
| **缺少负面过滤** | 只加分不减分（减分项太少），烂股也能混入满分 |

### 4.2 解决方案：三层漏斗式筛选

```
第一层：基础门槛（宽进） — J≤14 + 白>黄 + 涨跌幅±2% + 振幅<7%
    ↓ 过滤掉80%无关股票
第二层：路径质量评分（精筛） — 5特征评分 + 连续型微调分
    ↓ 保留评分≥80的候选
第三层：图形区分度排序（精选） — 新增区分特征，打破同分僵局
    ↓ 取TOP20
```

### 4.3 第三层需要的"区分特征"

这些是当前5特征体系完全缺失的，也是打破同分僵局的关键：

| 区分特征 | 作用 | 来源 |
|---------|------|------|
| **蜈蚣图检测** | 排除K线形态混乱的烂股（投研罗盘已有） | 5因子一票否决 |
| **沙漏审美** | 对图形质量连续打分，同分时按沙漏分排序（投研罗盘已有） | 5因子0-100分 |
| **麒麟阶段** | 识别庄家阶段，只选吸筹/回落期（投研罗盘已有） | 4阶段分类 |
| **MDC置信度** | 多维度置信度0.1~0.98，打破同分（投研罗盘已有） | 7层打分 |
| **连续型J值** | J=-13比J=13得分更高，而不是都≤14=通过 | 连续评分 |
| **连续型量能斜率** | 缩量趋势的斜率越陡越好，不是只看量比阈值 | 连续评分 |
| **路径分形相似度** | 计算候选与完美案例的DTW/余弦相似度 | 图形匹配 |

### 4.4 具体门槛建议

基于10个完美案例的特征值分布：

| 条件 | 门槛 | 依据 |
|------|------|------|
| J值 | ≤14 | 10个案例最大J=13.72（野马电池） |
| 白线>黄线 | 必须 | 10个案例全部满足 |
| 涨跌幅 | -2%~+2% | 10个案例全部满足 |
| 振幅 | <7% | 10个案例全部满足 |
| 5特征评分 | ≥75 | 10个案例最低74分（昂利康） |
| **蜈蚣图** | <60分 | 排除K线混乱的烂股 |
| **沙漏审美** | ≥40分 | 图形质量底线 |
| **MDC置信度** | ≥0.3 | 多维验证通过 |
| **排序依据** | 5特征分×0.5 + 沙漏分×0.3 + MDC×0.2 | 综合排序打破同分 |

## 五、结论

### 能否选出10个完美案例？

**纯靠调阈值：不能。** 这是一个数学上的不可能——放宽阈值让10个都通过，必然导致大量非完美股票也通过，完美案例被淹没。

**需要改变评分方法：可以。** 方案是：

1. **基础门槛用宽阈值**（J≤14 + 白>黄 + 5特征≥75）— 让10个完美案例都进门
2. **排序用连续型区分特征**（沙漏审美 + MDC置信度 + 蜈蚣图排除）— 在满分级别打破僵局
3. **取综合分TOP20** — 完美案例因为图形质量高、MDC置信度高，排名靠前

这正是投研罗盘的独有优势（蜈蚣图/沙漏/MDC）可以发挥作用的地方——不是替代Z哥的5特征评分，而是作为**第二层排序依据**，在5特征同分时用图形质量区分优劣。

### 一句话总结

> **5特征决定"能不能进门"，蜈蚣图/沙漏/MDC决定"排在第几名"。**
