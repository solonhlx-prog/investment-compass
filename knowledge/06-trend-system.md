# 双线趋势系统 · 06-trend-system

> 从 BBI 单线升级的右侧趋势系统。回答三个问题：主力在不在？洗盘还是转势？什么时候必须走？

---

## 一、双线定义

| 线 | 名称 | 公式 | 功能 |
|----|------|------|------|
| **白线** | 短期成本线 | `EMA(EMA(CLOSE, 10), 10)` | 短期趋势锚点，流畅波段的止损警戒线 |
| **黄线** | 长期成本线 | `(MA14 + MA28 + MA57 + MA114) / 4` | 主力入场标志，极限洗盘位，「大哥线」 |

**黄线是 4 参数 BBI 变体**：周期比标准 BBI 更长（14/28/57/114），代表中长期持仓成本。白线破黄线 = 短期成本跌破长期成本 = 主力可能已离场。

---

## 二、核心原则

| 原则 | 含义 | 操作含义 |
|------|------|---------|
| **白线 > 黄线** | 主力牵着牛绳 | 下跌大概率是洗盘，回调就是 B1 机会 |
| **白线 < 黄线** | 牛绳断了 | 上涨大概率是反弹，只卖不买 |
| **两线之间** | 「碗」——高容错率区间 | 不用等 B1，回调到碗底就可考虑 |
| **白线死叉黄线** | 短期趋势确认走坏 | **最后的离场时机，走错也要走** |
| **跌破白线** | 短期走弱 | 没利润垫就走；有 10%+ 利润可扛到黄线 |
| **跌破黄线** | 主力防线失守 | 牛股可以跌破白线，**绝不能跌破黄线** |

---

## 三、三道防线（分级止损）

| 级别 | 信号 | 操作 |
|------|------|------|
| **第一道** | 收盘跌破白线 | 没利润垫 → 减半清仓；有 10%+ 利润 → 可扛到黄线 |
| **第二道** | 收盘跌破黄线 | 最多忍一根 K 线，次日不收 = 全清 |
| **第三道** | 白线死叉黄线 | **无条件清仓**，走错也要走，最后防线 |

**在牛熊不同市场阶段的切换**：
- 左侧/熊市 → 用 BBI 线（敏感止损，不留恋）
- 右侧/牛市 → 用黄白线（稳拿波段，不被牛甩飞）
- 震荡市 → 黄白线走平缠绕 → 区间操作，不看趋势

---

## 四、五种玩法

| 玩法 | 条件 | 操作 | 交易价值 |
|------|------|------|---------|
| **玩法一：金叉入场** | 白线金叉黄线 + B1 回踩确认 | 入场，吃完整波段 | ⭐⭐⭐ |
| **玩法二：死叉离场** | 白线死叉黄线 | 最后的离场时机，走错也要走 | — |
| **玩法三：极限买点** | 白线死叉黄线多 + N 型不破 + J 大负值 | **极限买点**。大盘票成功率更高，港股也适用 | ⭐⭐⭐⭐⭐ |
| **玩法四：白黄区间** | N 型上涨中，价格在碗内 | 容错率相对较高，**跳出 B1 思维**，不用非等 J 到负 | ⭐⭐⭐⭐ |
| **玩法五：放量回踩** | 放量金叉后缩量回踩黄线，不破 | 连续拉升前的**最后震仓动作**，交易价值最大 | ⭐⭐⭐⭐⭐ |

> **玩法五（灾后重建）** = 放量金叉 → 缩量回踩黄线 → 不破 → 最高评级的入场信号。

---

## 五、循环操作法

```
B1 买入（白线 > 黄线，且 J 到负）
  → 横盘/温和上涨拿住
  → 白黄区间（碗内）考虑加仓（不倒金字塔）
  → 加速后放飞（减半锁利）
  → 跌破白线卖出
  → 观察白黄区间建仓价值
  → 重新循环 B1
```

**关键**：只在白线 > 黄线时做 B1。白线 < 黄线时，任何 B1 都要降级为"赌反弹"而非"做波段"。

---

## 六、牛绳理论

- **白线在黄线之上** = 主力牵着牛绳，任何下跌都是洗盘——「没利空消息的暴跌不卖，有本事打我止损」
- **白线在黄线之下** = 牛绳断了，任何上涨都是反弹——「反弹不是反转，仓位要轻，止损要近」
- 用多空双线判断趋势真假：K 线骗人，双线不骗人

---

## 七、碗（白黄之间的区域）

白黄线之间的区域叫「碗」。碗越大，容错率越高。

- **碗大** → 不需要等 J 到 -10，回调到碗底就可以考虑分批进
- **碗小** → 白线紧贴黄线 = 随时可能死叉 → 必须等标准 B1
- **碗在收敛** → 白线向黄线靠拢 = 变盘临界点，放量突破方向就是新趋势

---

## 八、牛熊版本选择

| 市场阶段 | 用哪套 | 理由 |
|---------|--------|------|
| **熊市末期 / 震荡市** | BBI 单线 | 敏感度高，跌破即走，不被阴跌套牢 |
| **牛市第二阶段** | 黄白双线 | BBI 太敏感会被「牛甩飞」，双线更稳 |
| **白线 > 黄线** | 按双线规则做多 | 信号置信度高 |
| **白线 < 黄线** | BBI 单线止损，不做多 | 双线已给出空头结论 |

---

## 九、迪迪图（大哥入场信号）

单根 K 线放量突破黄线——这是大哥进场的标准图形，表示主力开始画图。

**确认条件**：放量 > 前 5 日均量 2 倍 + 收盘站上黄线 + 次日不跌回。

---

## 十、局限

- 横盘震荡时双线走平缠绕，信号失效——此时换 BBI 单线或区间操作
- 对 T+0 市场（港股/美股）不适用，双线是为 A 股 T+1+涨跌停制度设计的
- 需要日线级别数据，60 分钟以下周期信号噪音大

---

## 十一、双线系统与 B1 选股的集成

B1 扫描出候选票后，**必须叠加双线判断**：

| 双线状态 | B1 处理方式 |
|---------|------------|
| **白线 > 黄线 + 碗大** | 优先关注，B1 确认后可入场 |
| **白线 > 黄线 + 碗小/收敛** | 等标准 B1（J≤-10 + 缩量 + 收盘十字星） |
| **白线 = 黄线（走平）** | 按震荡处理，轻仓 B1 |
| **白线 < 黄线（死叉态）** | **B1 信号作废**，不碰 |
