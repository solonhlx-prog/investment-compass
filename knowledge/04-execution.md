# 执行模块 · 04-execution

> 波段纪律法：低吸（B1）→ 半仓锁利 → 窄止损 → 不追高 → 积小胜为大胜。

---

## 一、波段纪律法六步 SOP

| 步骤 | 内容 | 规则 |
|------|------|------|
| **1. 择时** | 看大盘概率 | 活跃市值 +4% 以上才干活，-2.3% 以下不进场 |
| **2. 选股** | 五维筛选 | 政策支持 + 行业前景 + 龙头业绩 + 资金流入 + 估值合理 |
| **3. 等B1** | KDJ超卖信号 | KDJ(9,3,3)，J ≤ 12（最好负值）+ N型上移 + 缩量回调 |
| **4. 设止损** | 收盘价止损 | 入场K线最低价 / N结构前低。看收盘价，不看盘中 |
| **5. 止盈** | 放飞锁利 | 站上BBI后连续两根中/大阳线 → 减半。涨了先卖一半，锁住利润降低成本 |
| **6. 离场** | BBI两日破位 | 收盘价连续两天跌破BBI → 清仓。横盘不动也走 |

**核心约束**：每周最多交易 2 次。不要频繁操作。

## 二、B1 / B2 / B3 三重确认体系

三重保险，波段成立概率最大：

| 买点 | 含义 | 入场条件 | 仓位 |
|------|------|---------|------|
| **B1** | 回调波首次买点 | J ≤ -10、缩量回调、不破60日线/BBI黄线 | 分批慢买 |
| **B2** | 趋势确认买点 | 一阳穿多线、放量长阳、B1后3日内涨≥4%、KDJ钩<55 | 舒服买 |
| **B3** | 加速确认买点 | B2后出现十字星/小阴线、平开一致 | 当天干，次日不涨就走 |

**胜率排名**：B1 > B2 > B3。**确定性排名**：B3 > B2 > B1。

**核心原则**：B1 后必有 B2 确认，B2 后应有 B3 确认。新手先学会 B1。

### B1 量化筛选管线（P0 移植：MDC 七层验证 + 硬过滤）

投研罗盘 `scripts/screening.py` 将 B1 选股固化为可复用的量化管线，TT 模式（本地 Parquet 数据）下自动运行。全部阈值集中在 `config.B1_PARAMS`，支持 `.env` 覆盖。

**管线顺序**（任一环节不通过即淘汰）：
1. **基础条件过滤**：股价 > 3 元；J ≤ 阈值（默认 -8）；近 10 日 J 钝化排除；量比 < 0.85；乖离 MA20 ≤ -3%；10 日跌幅 ≥ -25%（排除崩盘）；双线白 > 黄；非绿砖状态（近 4 日阴线 < 上限）。
2. **硬过滤一：蜈蚣图**（一票否决）。堆量不涨、影线交替、无呼吸节奏的烂股。五大因子（长上影/长下影/十字星/量能无规律/价格无趋势）各 0-20 分，**总分 ≥ 60 直接排除**。
3. **硬过滤二：沙漏审美**（一票否决）。图形审美五因子（缩量收敛/枢轴邻近/量能斜率/均线结构/事件风险）各 0-20 分，**总分 < 50（默认）直接排除**。≥80 为完美图形。
4. **MDC 七层多维验证**（决定置信度 0.1~0.98）：
   - 麒麟阶段背景：吸筹 +20% / 回落 +10% / **派发 -30%（一票否决）**
   - 触及布林下轨（超跌）+15%
   - 主力大单净流入 +10%
   - RSI6 < 25（极端超卖）+5%
   - ADX > 40（动能竭尽）+10%
   - 缩量回调 +0.1 基础置信度
5. **打分输出**：基础分 + MDC 置信度映射分，按分数降序返回 TOP 候选。

> 数据不足时逐层降级：麒麟（需 ≥60 日）、蜈蚣图/沙漏（需 ≥20 日）、ADX（需 ≥15 日）在窗口不够时自动跳过该层，绝不报错。websearch 模式无本地数据时整管线优雅跳过。

## 三、B1 入场三问

B1 不是见信号就干。入场前必问：

| 问题 | 不合格的表现 |
|------|-------------|
| **离黄线近不近？止损能否设？** | 在白线和黄线中间飘着，往下还有 10-15% |
| **上方有没有标准压力？** | 顶部有 S1、阶梯量等压力，反弹空间不足 |
| **是新起点还是余震？** | 前面已涨完，A杀后赌反弹 |

有瑕疵就不干。A股 5000 多只票，没必要对别扭的图自我说服。

### B1 选股必加双线过滤（06-trend-system §十一）

B1 扫描出候选票后，必须叠加双线系统判断：

| 双线状态 | B1 处理方式 |
|---------|------------|
| **白线 > 黄线 + 碗大** | 优先关注，B1 确认后可入场 |
| **白线 > 黄线 + 碗小/收敛** | 等标准 B1（J≤-10 + 缩量 + 收盘十字星） |
| **白线 = 黄线（走平）** | 按震荡处理，轻仓 B1 |
| **白线 < 黄线（死叉态）** | **B1 信号作废**，不碰 |

**核心逻辑**：白线 < 黄线 = 主力不在。主力不在的票，J 值再低也是接飞刀，不是做波段。

参考 → `knowledge/06-trend-system.md`

## 四、四块砖法则（情绪周期）

A股短期每 4 天一个情绪循环：

| 状态 | 规则 |
|------|------|
| **红砖上涨** | 第1块确认入场 → 持有 → 第4块走完至少减半 |
| **绿砖下跌** | 绝不抄底 → 先数满4块 → 翻红且不创新低才轻仓试错 |
| **第4砖加速** | 如果第4块红砖加速上涨，下一个下跌周期第4块绿砖也会加速下跌 |
| **时间纪律** | 买入后3天不涨就走，最多拿4天 |

## 五、量比战法（开盘择时）

B1 入场后次日的量比决定了当天怎么操作：

| 量比 | 行情 | 操作 |
|------|------|------|
| **< 10** | 正常/震荡 | 慢买，逢低吸纳 |
| **> 20 + 涨** | 攻击日 | 立即买 |
| **> 20 + 跌** | 出货日 | 观望 |
| **10-20 + 量价齐升** | 单向拉升 | 买 |
| **10-20 + 低开-2%以下** | 弱势 | 不接飞刀 |

## 六、超级 B1（洗盘结束信号）

正常缩量回调中，突然来一根放量下杀阴线（非跌停）。

- 之后缩量 + J 依然大负值 + 出现反转小十字星 = 超级 B1
- **入场**：反转十字星出现次日
- **止损**：放量破位阴线的最低价
- **铁律**：只赌一次

## 七、伪突破 / 一日游识别

阴跌中突然出现一根放量阳线，量放大 2-3 倍，但第二天：

| 次日表现 | 判断 | 操作 |
|---------|------|------|
| 缩量回落（量 < 前日 50%） | **一日游骗线** | 不进，此信号作废 |
| 继续放量 + 站稳 BBI | 真突破 | 右侧确认后再进 |
| 放量但价格回落 | 出货 | 远离 |

**识别口诀**：量放了三倍只涨一天，后面没人跟 = 抄底资金一日游 = 骗线。阴跌中的反弹，十次有八次是假的。

**确认真突破的三个条件**（三者缺一不可）：
1. 次日继续放量（量不低于前日 70%）
2. 收盘站稳 BBI
3. 第三天不跌回

缺任何一个条件 = 按伪突破处理。

## 八、阴跌 B1 特殊规则

空头排列 + 持续阴跌的票，B1 条件比正常更严格：

| 条件 | 正常 B1 | 阴跌 B1 |
|------|---------|---------|
| J 值 | ≤ -10 | **≤ -5**（确认负值，防钝化误判） |
| 量比 | 无硬性要求 | **< 0.6**（极致缩量才考虑） |
| 止跌信号 | N 型结构 | **十字星 / 小阳线 / 下影线收阳** |
| 仓位 | 分批慢买 | **≤ 1/3 仓位** |
| 止损 | 入场 K 线最低价 | **进场当日最低点下方 2 分钱** |
| 均线 | 不破 60 日线 | 可以不看 MA60（大概率已破），但 **必须等 BBI 压力距离在 5% 以内** |

**阴跌 B1 的铁律**：量不够缩（量比 >0.6）+ J 没到负 = 绝不进场。阴跌没缩到位就进，等于半路上车。

## 十、量能对比规则（反弹 or 反转）

同一指数/个股，对比相邻涨跌日的成交额：

| 量价关系 | 含义 | 操作 |
|---------|------|------|
| **放量涨 + 缩量跌** | 资金进场，回调是洗盘 | 持有/加仓 |
| **放量跌 + 缩量涨** | 反弹非反转 | 减仓/不追 |
| **放量跌 + 放量涨** | 多空交战激烈 | 等方向确认 |
| **缩量跌 + 缩量涨** | 无人关注 | 不参与 |

**核心规则**：反弹日的量如果小于暴跌日的量 → 这是反弹，不是反转。可以做，但仓位要轻，止损要近。

## 十一、场景化操作模板

### 情况 A：仓位重（>70%）

- 反弹减仓。弹得猛的时候正好给你跑路
- 不是清仓，是把动态仓降下来
- 底仓（低波红利/稀缺资产）不动，动态仓该走的走

### 情况 B：仓位轻（<30%）

- 关键点位（如4000点）附近是低吸窗口
- 不是一天全买——每天筛 1-2 只合格票慢慢加
- B1 位置的就买 B1，没信号的就不买
- 不要因为空仓焦虑而乱买

### 情况 C：空仓观望

- 继续观望
- 等放量站稳 MA60 的信号出来再做右侧
- 不做左侧抄底——你接不住飞刀

### 情况 D：震荡市（大多数时间）

- 仓位 ≤50%，单票 ≤5%
- 底仓不动，动态仓 B1 买 + 放飞卖
- 不追板块热点（消息驱动来得快去得也快）

## 十二、三句铁律

1. **择时永远第一**：天好再干活，天不好就休息。
2. **放飞一定要养成习惯**：放飞不是怂，是把利润先揣兜里。
3. **没涨和盈转亏要优先**：市场没证明你对，你就先走。先防守，再进攻。
