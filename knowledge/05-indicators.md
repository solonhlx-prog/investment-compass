# 指标手册 · 05-indicators

> 核心技术指标的判定公式与使用场景。

---

## 一、KDJ（主要买入信号源）

**参数**：(9, 3, 3)

**K 值** = `SMA(RSV, 3, 1)`
**D 值** = `SMA(K, 3, 1)`
**J 值** = `3*K - 2*D`

| J 值区间 | 含义 | 操作 |
|---------|------|------|
| **J ≤ -10** | 超卖信号，B1 买点核心条件 | 配合缩量+N型结构，入场 |
| **-10 < J < 0** | 偏弱，可关注 | 等进一步确认 |
| **J > 0** | 不适合 B1 操作 | 不是倒车接人，是接飞刀 |
| **J 死叉** | J 在 K 和 D 之下（状态，非瞬间） | 卖出参考信号 |

**J 值钝化**：J 在超卖区低位横着不涨（如 J 从 5→-2→-8 慢慢滑），不是超卖信号，是趋势还在延续。空头排列中追 B1，J 钝化阶段十条腿都不够断的。必须等 J 确确实实掉进负值区 + 第二天有止跌信号（十字星/小阳线/极致缩量）才算数。

**注意**：J 死叉是状态不是瞬间——只要 J 在 K 和 D 之下就算死叉，不是下穿那一瞬间才算。

## 二、MACD（趋势确认与背离）

**DIF** = `EMA(CLOSE, 12) - EMA(CLOSE, 26)`
**DEA** = `EMA(DIF, 9)`
**MACD 柱** = `2*(DIF - DEA)`

| 信号 | 判定 | 操作 |
|------|------|------|
| **零轴以上 DIF** | 多头区域 | 可以做多 |
| **零轴以下 DIF** | 空头区域 | 谨慎，不做多 |
| **金叉**（DIF上穿DEA） | 多头信号 | 配合量能确认 |
| **死叉**（DIF下穿DEA） | 空头信号 | 减仓/清仓 |
| **顶背离**（价创新高，MACD柱未新高） | S2 逃顶信号 | 趋势反转，必走 |
| **底背离**（价创新低，MACD柱未新低） | 底部信号 | 关注，等 B1 确认 |

**核心认知**：MACD 的滞后性是灵魂。不是提前预测，是趋势确认后才动手。

## 三、BBI（多空基准线）

**BBI** = `(MA3 + MA6 + MA12 + MA24) / 4`

| 位置关系 | 含义 | 操作 |
|---------|------|------|
| **价格在 BBI 之上** | 多头趋势 | 持有/做多 |
| **价格在 BBI 之下** | 空头趋势 | 不建新仓 |
| **连续两日跌破 BBI** | 止损信号 | 清仓离场 |
| **BBI 走平** | 震荡 | 区间操作 |

**用法**：B1 入场后，站上 BBI 出现连续两根中/大阳线 → 减半锁利。横盘但未破 BBI → 继续等。

## 四、量比（开盘择时）

**量比** = 当日开盘成交量 / 前5日平均每分钟成交量

| 量比区间 | 行情类型 | 操作 |
|---------|---------|------|
| **< 10** | 正常/震荡 | 慢买/观望 |
| **10-20 + 涨** | 偏强 | 可买 |
| **10-20 + 跌** | 偏弱 | 观望 |
| **> 20 + 涨** | 攻击日 | 立即买 |
| **> 20 + 跌** | 出货日 | 观望，除非盘中突破日内高点 |

## 五、成交量形态判定

| 形态 | 特征 | 含义 |
|------|------|------|
| **红肥绿瘦** | 阳量持续大于阴量 | 底部吸筹，积极信号 |
| **绿肥红瘦** | 阴量持续大于阳量 | 顶部出货，警惕 |
| **倍量柱** | 当天成交量 > 前日 2 倍 | 主力动作，关键信号 |
| **缩量回调** | 下跌过程缩量至前高的 50% 以下 | 洗盘结束信号，等 B1 |
| **蜈蚣图** | 堆量不涨，影线交错 | 没有交易价值，直接排除 |

## 六、累计换手率（B1 选股补丁）

| 补丁 | 条件 | 用途 |
|------|------|------|
| **周线多头** | MA55 > MA144 > MA233，趋势向上 | 过滤下跌趋势票 |
| **累计换手 < 38%** | 前三根中大阳线累计换手 < 38% | 排除筹码发散/主力跑路的票 |

换手率过高 = 筹码已经发散 = 主力跑了。累计换手 38% 是警戒线。

## 七、均线系统与排列判定

| 均线 | 周期 | 用途 |
|------|------|------|
| MA5 | 5日 | 超短线参考 |
| MA10 | 10日 | 短线趋势 |
| MA20 | 20日（月线） | 短线趋势 |
| MA60 | 60日（季线） | 中线趋势，重要的支撑/压力 |
| MA120 | 120日（半年线） | 牛熊分界 |
| MA250 | 250日（年线） | 长期牛熊分界 |

### 均线排列判定

| 排列状态 | 条件 | 含义 | 操作 |
|---------|------|------|------|
| **多头排列** | MA5 > MA10 > MA20 > MA60 | 强势上升趋势 | 做多，回调买 |
| **空头排列** | MA5 < MA10 < MA20 < MA60 | 强势下跌趋势 | 不做多，等反转 |
| **交叉缠绕** | 均线上下交错 | 方向不明 | 区间操作/观望 |

**关键用法**：空头排列 + MACD零轴下方死叉 + 价格在所有均线之下 = **教科书级熊势**。这种情况下任何 B1 都要等极端缩量和 J 确凿负值。

## 八、乖离率（偏离度）

**乖离率(MA20)** = `(CLOSE - MA20) / MA20 * 100%`

| 乖离 | 含义 | 操作 |
|------|------|------|
| **-5% ~ -10%** | 超跌偏离 | 关注，可能有技术性反弹 |
| **> -5%** | 正常范围 | 按原趋势判断 |
| **> +10%** | 上涨过度 | 警惕回调 |

**注意**：乖离大 ≠ 立即反弹。在空头排列中乖离可以继续扩大。乖离只是"偏离过大"，不是"到了就该弹"。

## 九、基本面速查（个股分析必含）

分析个股时必须调取以下三项：

| 指标 | 含义 | 判定参考 |
|------|------|---------|
| **PE** | 市盈率 | <15=低估值，>50=高估值（行业差异大） |
| **PB** | 市净率 | <1=破净，建筑/银行等行业常见 |
| **负债率** | 资产负债率 | >70%=高杠杆，建筑/地产行业通病 |

**行业定性**：基本面数据须结合行业判断。如建筑股 PB<1、负债率高是行业常态，不代表"便宜就该买"——没故事、没预期差的行业，便宜不是买入理由。
